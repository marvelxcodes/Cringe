
# About this Project

**Cringe is a full feldged Shorts based Social media WebApp with features like:**
    - 60s Vid Posts
    - Swipe up to Skip
    - Comment Feature
    - Like and Heart Feature
    - User could post Videos

# Project Stack

## I've used the following tech in the Project

**Frontend**
    - [React JS](https://reactjs.org/)
    - [Next JS](https://nextjs.org/)

**Backend**
    - [Sanity CMS](https://sanity.io)
    - [Node JS](https://nodejs.org/)

**Styling**
    - [Sass](https://sass-lang.com/)
    - [Tailwind CSS](https://tailwindcss.com/)

**Libraries**
    - [Bootstrap Icons](https://icons.getbootstrap.com/)

**Tools**
    - [Typescript](https://www.typescriptlang.org/)

# Clone this Repo

**Run the following commands in Terminal**
git clone "https://github.com/marvelxcodes/Cringe"
cd Cringe
npm install
